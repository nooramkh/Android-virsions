# Android Versions App

تطبيق أندرويد بسيط يعرض قائمة بإصدارات نظام الأندرويد المختلفة باستخدام أداة RecyclerView.

## الميزات (Features):
* عرض قائمة عمودية تحتوي على 12 إصداراً من إصدارات أندرويد.
* استخدام CardView لتصميم البطاقات بشكل مرتب.
* عرض صورة الشعار، واسم الإصدار، ورقم الإصدار داخل كل بطاقة.
* الألوان المتبادلة: تم برمجة خلفيات البطاقات لتظهر بألوان متبادلة (صف أبيض وصف رمادي) لتسهيل القراءة.
* التفاعل: عند النقر على أي بطاقة، تظهر رسالة Toast للمستخدم توضح اسم ورقم الإصدار الذي تم اختياره.
